# bme280-exporter

## Hardware

- [Raspberry Pi 3](https://www.raspberrypi.org/products/raspberry-pi-3-model-b/)
- [AE-BME280](http://akizukidenshi.com/catalog/g/gK-09421/)

### Pins

AE-BME280  | Raspberry Pi 3
---------- | --------------
VDD (`#1`) |   3.3V (`#1`)
GND (`#2`) | Ground (`#6`)
SDI (`#4`) | GPIO 1 (`#3`)
SDO (`#5`) | Ground (`#9`)
SDK (`#6`) | GPIO 2 (`#5`)
